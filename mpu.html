Cadastro de MPU no SISEG


  1. Link do sistema SISEG https://emergencia.sesdec.ro.gov.br/sade/sade/public/index.php/login

    Link do sistema SISEG Treinamentohttps://treinamento.emergencia.sesdec.ro.gov.br/sade/sade/public/index.php/login

  2. Utilizando a senha do perfil MPU acessar o SISEG

  3. Após acessar o menu CADASTROS - Opção Medida Protetiva de Urgência

    4. * Realizar a pesquisar pelo PROTOCOLO da ocorrência.

  4. Após localizar a ocorrência e clicar no ícone Gerar MPU

  5. Realizar o preenchimento de todos os campos da tela seguinte, com os dados da Medida Protetiva e anexar o arquivo em PDF.

    6. * É possível anexar mais de um arquivo PDF para a mesma Medida Protetiva, no entando tem que salvar primeiro as informações e depois acessar novamente a Medida para anexar um segundo arquivo PDF.


    Nos campos destinados as informações para o Agressor:
    Restrição - Grau de Risco - Requerido notificado - Condição do Agressor e Mandado de Prisão Agressor.
    Deve-se selecionar a opção correspondente na caixa de seleção
    e após clicar em adicionar

    Salvar as informações.


    Editar ocorrência para MPU no SISEG

  6. Caso não aparecer o nome do Agressor

    7. Deve-se retornar a tela anterior “Cadastro Medida Protetiva Urgência - MPU” para fazer a edição da OCORRÊNCIA



    Na tela “Cadastro Medida Protetiva Urgência - MPU” localizar a OCORRÊNCIA e clicar no ícone

  7. Tela de edição da OCORRÊNCIA

    8. 1º Alterar o Código de Encerramento para um diferente do que está salvo podendo ser o “CF-24-Inválido” para permitir edição dos dados da OCORRÊNCIA.

    2º Acessar o Menu ENVOLVIDOS

    Rolar a tela até o final

    Clicar no ícone
    correspondente ao Autor ou Vítima para fazer a edição dos dados dos Envolvidos.

    Após acrescentar os dados necessárias rolar a tela até o final e salvar as informações.

    Pronto foram adicionado as informações dos Envolvidos, fechar a janela no X.
    *** Importante não é necessário retornar para a aba Dados da Ocorrência ***.

    MEDIDAS PROTETIVAS QUE NÃO POSSUEM OCORRÊNCIA

    Deve-se criar uma ocorrência para que se possa anexar uma medida protetiva e liberar a programação operacional das visitas.
    Com a senha do perfil DESPACHADOR acessar o SISEG
    Clicar no menu “Gerar Ocorrência”


    Fazer o prenchimento dos seguinte campos
    Descrição = Medida Protetiva de Urgência - MPU
    Agência = PM
    Natureza = lesão Corporal
    Marcar os campos de nível de risco =
    Classificação = Ocorrência
    Local = Direto no Quartel
    Nome = Analista
    Endereço = da vítima
    Salvar informações

    ENCERRAMENTO
    Após cadastrar a ocorrência ir para o menu Acompanhar Ocorrências, localizar o protocolo correspondente e clicar em "Alterar para pendente"

    Após acessar o menu Alterar para pendente

    clicar no ícone para fazer o encerramento.

    DADOS DA OCORRÊNCIA - CAMPOS
    Código de Encerramento = Usar o CF 01 - Comunicação de Ocorrência Policial;
    Relato policial = usar texto padrão “ocorrência já registrada em outro sistema para cadastro de MPU”;
    Natureza = selecionar lesão corporal;
    Após clicar no ícone

    ENVOLVIDOS - CAMPOS
    Preencher primeiro as informações do autor.
    Nome Completo:
    Data de nascimento:
    Telefone:
    RG: / UF Órgão Expedidor: se possuir a informação
    Tipo de Envolvimento: Autor
    Fato Vinculado: lesão corporal
    Violência doméstica: marcar a opção sim

    O campo vínculo com o Agressor só é preenchimento para a Vítima.

    Local de condução = outro locais
    Prisão/apreensão:

    Campos obrigatórios
    Sexo:
    Raça cor da pele:
    Endereço:
    CPF: deve ser preenchido para habilitar o cadstrp da MPU

    Após clicar em Adicionar e voltar ao inicio da página para adicionar a Vítima preenchendo os mesmos campos do autor.
    Acrescentando a informação do vínculo com o Agressor.

    Com a vítima adicionada na Ocorrência clicar na aba "Dados da Ocorrência" para finalizar clicando no ícone no canto inferior direito.

    Com isso a ocorrência saíra da tela de ocorrências em andamento e estará disponível para cadastro da MPU no perfil MPU.

    PROGRAMAÇÃO OPERACIONAL

    Para acessar Link de Vídeo da instrução sobre a Programação Operacional: "Clique Aqui"

    Necessário ter o número da MPU que vai ser realizada a visita. No SISEG entrar no menu “CADASTROS” - “Programação Operacional”

    Rolar a página até o final e clicar na opção “NOVO”

    Preencher todos os campos
    Descrição: Colocar o nome da Patrulha “Patrulha Maria da Penha 1º BPM”;
    Medida Protetiva de Urgência: Pesquisar para qual MPU será realizada a visita;
    Local da Programação operacional: Área de atuação da patrulha;
    Tipo de Operação: Visita MPU;
    Data ínicio: / Data fim:
    Hora inicial: / Hora final:
    Salvar

    Ao salvar retornar-se para a tela de Pesquisa Programação Operacional
    Deve-se localizar programação para gerenciar a guarnição de atendimento.


    1º ícone - Gerenciar guarnição
    2º ícone - Visualizar observações
    3º ícone - Visualizar anexos / observações
    4º ícone - Visualizar detalhes
    5º ícone - Editar programação
    6º ícone - Exclui a programação

    Vamos clicar no ícone para empenhar guarnição para realização da visita a MPU.
    Na tela seguinte pesquisa-se a guarnição podendo se por “VOUCHER” ou pelo “número da guarnição”.

    Clicar no ícone gerenciar tipo de alocação.

    Definir as informações e salvar.

    Após salvar já está alocada uma guarnição e vai aparecer na tela as informações.

    Com as seguintes opções
    1º ícone - Informar chegada
    2º ícone - Desalocar guarnição
    3º ícone - Programação não executada
    4º ícone - Ativar dedicação exclusiva - nesse caso a patrulha não recebe solicitações do CICC
    Não sendo necessário fazer alterações.
